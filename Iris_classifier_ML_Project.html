Introduction¶

To create a demo machine learning model that takes four measurements from the flowers (sepal length, sepal width, petal length, and petal width) and identifies the species based on those measurements alone. The dataset that has been used for this exercise is famous iris dataset about three species of Iris flower. The categories being 'Iris-setosa','Iris-versicolor' and 'Iris-virginica'. We have to develop a machine learning model that can identify a given flower,given its measurements from the above three flowers.

Let us have a look at our dataset and understand a bit about it.¶

# Defining a function to correct the 'class' column

def classfunction(x) :
    # Checking for wrong enteries and correcting them.
    if x =='Iris-setossa':
        x = 'Iris-setosa'
    elif x == 'versicolor':
        x = 'Iris-versicolor'
    else:
        pass
    return x

Parameter tuning¶

In [62]:
#Using grid search for parameter tuning
from sklearn.cross_validation import StratifiedKFold
from sklearn.grid_search import GridSearchCV

decision_tree_classifier = DecisionTreeClassifier()

parameter_grid = {'max_depth': [1, 2, 3, 4, 5],
                  'max_features': [1, 2, 3, 4]}

cross_validation = StratifiedKFold(all_classes, n_folds=10)

grid_search = GridSearchCV(decision_tree_classifier,
                           param_grid=parameter_grid,
                           cv=cross_validation)

grid_search.fit(all_inputs, all_classes)
print('Best score: {}'.format(grid_search.best_score_))
print('Best parameters: {}'.format(grid_search.best_params_))
